# 🛒 MegaMarket

E-ticaret pazaryeri platformu. Amazon/Hepsiburada tarzı.

## Özellikler
- 🏪 Çoklu satıcı desteği
- 🔍 Gelişmiş arama (Elasticsearch)
- 🛒 Sepet ve wishlist
- 💳 Ödeme entegrasyonu
- 📦 Sipariş takibi
- ⭐ Değerlendirme sistemi
- 🏷️ İndirim kuponları

## Teknolojiler
- Next.js 14
- Node.js + Express
- PostgreSQL + Redis
- Elasticsearch
- Stripe

## Kurulum
```bash
npm install
npm run dev
```
